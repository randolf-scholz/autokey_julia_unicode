# Documentation

Font family providers:

## Builtin font-familied

- `\mathrm`
- `\mathit`
- `\mathsf`
- `\mathbf`
- `\mathtt`
- `\mathcal`

extra packages required for:

- `\mathfrak`
- `\mathscr`
- `\mathbb`

## Bold symbols

If we want bold symbols, in particular also: greek+bold, italic+bold, etc. There are several ways to do this.
